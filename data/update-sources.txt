# Format:  <type>  <target>  <url>
#   type   = trusted | community
#            trusted   -> file is downloaded and REPLACES the target verbatim
#                         (use only for repos you control/trust; a .bak is kept).
#            community -> file is parsed, and only clean ".ext" entries are
#                         UNION-merged into extensions-auto.txt (never deletes
#                         anything, never overwrites your curated extensions.txt).
#   target = file name under data\  (or a logical name for community extensions)
#   url    = raw download URL
#
# Community lists are filtered hard: wildcard/garbage patterns (*.crypto*,
# *.*obleep, [ZipLOCK]*, hashes) and purely-numeric extensions (which collide
# with split-archive .001/.007 files) are dropped to avoid false positives.
#
# Lines starting with # are ignored. Add or remove sources freely.
